Overview

The Makeline has a collection of screens and lighting devices used to assist you in operation. The screens work independently to provide useful information about and control over the Makeline. Then, if an issue arises, the screens work together with the lights to support you in fault identification and resolution.

General Resolution Process

  1. A fault is indicated on the lights and/or screens.
    1. Automation continues
  2. Take necessary action to resolve the fault
  3. Confirm the fault is resolved by pressing the button on the Cabinet Screen
    1. If the fault can be addressed quickly, confirmation is given on the cabinet screen and automation resumes.
    2. If the fault persists, then the Makeline moves into a System Faulted state and the Makeline stops.
  4. Once the System Fault is addressed, the resolution is confirmed on the Lift Screen, and automation resumes.

Device Descriptions

Screens

Lift Screen

Description

The Lift Screen, is a 10” touch screen whose main purpose is to provide the you feedback as to the current state of the system, and facilitate state switching when appropriate.

Use case(s):

  1. System state status
  2. Entree assembly status
  3. System state control
  4. Lift control
  5. System-level fault resolution

Cabinet Screens

Description

Each cabinet on the Makeline has a touch screen to provide the current and target temperatures of the cabinet (except for the Denester and Lift), and facilitate fault resolution when applicable.

Use case(s):

Dispenser cabinets

  1. Display current cabinet temperature
  2. Set cabinet temperature
  3. Display ingredient supply level
  4. Fault information and resolution

Denester cabinets

  1. Bowl supply level indicator
  2. Fault information and resolution

Lift cabinets

  1. Item on lift information
  2. Manual control to raise lift
  3. Fault information and resolution

Lights

Counter Lights

Description

Each cabinet, at counter level, has a segmented ring of light that encircles each group of dispensers. They can be lighted in varied patterns and colors to communicate location and type of fault at hand.

Use case(s):

  1. Alert the user to a device-level fault
  2. Indicate to the user the specific physical location of a device fault

Cabinet Lights

Description

The Lift and Denester cabinets don’t possess counter lighting, but instead , have a bar of light located above the cabinet screen. They serve the same purpose as the counter lighting, notifying you when an issue arises.

Use case(s):

  1. Alert the user to a device-level fault
  2. Alert the user when an item is ready to raised on the Lift.
