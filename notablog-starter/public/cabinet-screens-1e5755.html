Cabinet Screens

test food bar

Overview

Each cabinet on the Makeline has a touch screen to provide the current and target temperatures of the cabinet (except for the Denester and Lift), and facilitate fault resolution when applicable.

User Interface

These are the basic UI components of the Cabinet Screen.

Default (Temp) UI Layout

  1. Temp up button
  2. Temp down button
  3. Current internal cabinet temperature

Faulted UI Layout

  1. Confirm resolved fault button
  2. Content title
  3. Temperature toggle button
  4. Fault name

Temperature

The default information displayed on the Cabinet Screen for all dispenser cabinets is the current internal temperature. You can adjust the temperature up and down by tapping on the up/down arrow buttons respectively.

Details

  1. Default state
  2. Adjusting temperature up
  3. Adjusting temperature down

Faults

When the Makeline detects an issue with the system, it will alert the user in a few ways, one of which is the Cabinet Screen.

The Cabinet Screen will show that there is a fault, an short description of the fault type, and provide a button to confirm the fault has been resolved by you.

Details

  1. Default faulted state
  2. Confirming fault is resolved
