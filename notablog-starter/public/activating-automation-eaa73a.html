Pausing Automation

There will be cases in which daily operators will need to pause the makeline for troubleshooting or other needs. They are able to accomplish this task by following the steps outlined below.

Pause

The operator is able to pause the makeline via the lift cabinet screen.

  1. Operator will pull down the menu bar on the lift screen and select Pause
  2. Operator will confirm the Pause action
  3. Lift screen will verify the transition into an paused state
  4. Makeline is now in a paused state

Operator will complete appropriate tasks before resuming automated operations

Resume

Once the operator is ready to transition back into an automated state, they can use the lift screen to move out of a paused state.

  1. Operator will select Resume Auto
  2. Cabinet screen will verify the transition into an automated state
  3. Operator will resume normal operations
