Special Cases

Incomplete Entree

When an entree has been partially completed, but must stop because of a System Fault, you are given an opportunity to manually move the partially lost entree out of the makeline so you can finish it manually.

Fully Lost Entree

When an entree falls of the conveyance system and spills into the bottom of the cabinet, it is considered fully lost. Similar to a partial lost entree, manual assembly is required. In this case, the entire entree must be manually created by you.
